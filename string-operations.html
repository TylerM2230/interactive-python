Understanding String Operations

An Interactive Guide (using Python examples)

1. What are Strings?

A string (str in Python) is a sequence of characters. Think of it as text data. Strings are enclosed in either single quotes ('...') or double quotes ("..."). They are immutable, meaning once a string is created, its contents cannot be directly changed (though you can create new strings based on existing ones).

name = 'Alice'
empty_string = ""

2. Combining & Repeating Strings

You can easily combine (concatenate) strings using the + operator and repeat strings using the * operator. 

first = "Hello"
second = " "
third = "There"

# Concatenation
full_greeting = first + second + third
print(full_greeting) # Output: Hello There

# Repetition
repeated_laugh = "Ha" * 3
print(repeated_laugh) # Output: HaHaHa

3. Accessing Parts of Strings

Since strings are sequences, you can access individual characters using indexing (starting from 0) and extract substrings using slicing.

  • Indexing [index]: Gets the character at a specific position. Negative indices count from the end (-1 is the last character).
  • Slicing [start:stop:step]: Extracts a portion of the string.
    • start: The index to begin extraction (inclusive, defaults to 0).
    • stop: The index to end extraction (exclusive, defaults to end of string).
    • step: The amount to jump between characters (defaults to 1). 
text = "Python"

# Indexing
print(text[0])    # Output: P (First character)
print(text[2])    # Output: t
print(text[-1])   # Output: n (Last character)

# Slicing
print(text[0:3])  # Output: Pyt (Chars at index 0, 1, 2)
print(text[2:])   # Output: thon (From index 2 to the end)
print(text[:4])   # Output: Pyth (From start up to index 4)
print(text[::2])  # Output: Pto (Every second character)
print(text[::-1]) # Output: nohtyP (Reverse the string)

4. Common String Methods

Strings have many useful built-in methods (functions attached to the string object) that return *new* strings (remember, strings are immutable).

  • .upper(): Returns a copy with all characters uppercase.
  • .lower(): Returns a copy with all characters lowercase.
  • .strip(): Returns a copy with leading/trailing whitespace removed.
  • .replace(old, new): Returns a copy with all occurrences of old replaced by new.
  • .find(substring): Returns the starting index of the first occurrence of substring (or -1 if not found).
  • .split(separator): Returns a list of substrings split by the separator (defaults to whitespace).
  • len(string): (This is a function, not a method) Returns the number of characters in the string. 
sentence = "  Hello Python World!  "

print(f"Original: '{sentence}'")
print(f"Length: {len(sentence)}")
print(f".upper(): '{sentence.upper()}'")
print(f".lower(): '{sentence.lower()}'")
print(f".strip(): '{sentence.strip()}'")
print(f".replace('o', '0'): '{sentence.replace('o', '0')}'")
print(f".find('Python'): {sentence.find('Python')}") # Index where 'Python' starts
print(f".split(' '): {sentence.split()}") # Splits by spaces
