Python's Random Module

1. Why Use Random Numbers?

Random numbers are key for simulations, games (dice rolls, card shuffles), statistics, and more. Python's random module helps generate these "pseudo-random" numbers (they seem random but are from a reproducible sequence if you set a "seed").

2. Importing and Common Functions

First, import: import random. Then use functions like:

  • random.random(): Float in [0.0, 1.0).
  • random.uniform(a, b): Float between a and b.
  • random.randint(a, b): Integer between a and b (inclusive).
  • random.choice(sequence): Random item from a list/tuple/string.
  • random.shuffle(list): Shuffles a list in-place.
  • random.seed(a): Initializes generator for reproducible results.

3. Interactive Random Generators

Try generating different types of random values!

Float (0.0 to < 1.0): random.random()

Float in Range: random.uniform(min, max)

Integer in Range: random.randint(min, max) (Dice Roll)

Choice from List: random.choice(items)

Shuffle List: random.shuffle(items)
