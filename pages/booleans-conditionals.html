Booleans & Conditionals

Making decisions in your code with True, False, and if statements.

1. What are Booleans?

A boolean is a data type that can only have one of two values: True or False. As we touched upon when discussing data types, booleans are like simple light switches: they can only be in one of two positions, ON (True) or OFF (False). They are fundamental for controlling the flow of a program by allowing you to make decisions. Think of them as answers to yes/no questions. 

is_game_over = False
has_permission = True

print(is_game_over)  # Output: False
print(type(has_permission)) # Output:

2. Comparison Operators

Comparison operators are used to compare two values. The result of a comparison is always a boolean value (True or False). Think of them as asking specific questions about the relationship between two values; the answer you get back is always a simple `True` or `False`.

  • == : Equal to (e.g., 5 == 5 is True) - Are these two values *exactly the same*?
  • != : Not equal to (e.g., 5 != 3 is True) - Are these two values *different*?
  • > : Greater than (e.g., 10 > 5 is True) - Is the first value *more* than the second?
  • < : Less than (e.g., 3 < 7 is True) - Is the first value *less* than the second?
  • >= : Greater than or equal to (e.g., 5 >= 5 is True) - Is the first value *more than or the same as* the second?
  • <= : Less than or equal to (e.g., 4 <= 5 is True) - Is the first value *less than or the same as* the second?
age = 18
required_age = 21

can_enter_club = age >= required_age  # This will be False
print(can_enter_club)

score = 100
target_score = 100
is_target_reached = score == target_score # This will be True
print(is_target_reached)

3. Conditional Statements: if, elif, else

Conditional statements allow you to execute certain blocks of code only if a specific condition is met (i.e., evaluates to True). These are like choosing different paths on a journey based on signs you encounter. The if, elif, and else keywords are your signposts, guiding the flow of your program.

The if Statement:

The simplest conditional. The code inside an if block only runs if the condition is True. It's like a gate with a condition (e.g., 'Is the player's score above 1000?'). If the answer is True, the gate opens, and the program executes the code inside. If False, that code block is skipped. 

temperature = 30
if temperature > 25: # Sign: 'If temperature is above 25'
    print("It's a hot day!") # Action: Announce hot day.

if temperature < 10:
    print("It's a cold day!") # This condition is False, so this is skipped.

The else Statement:

Used with if to provide an alternative block of code to execute if the if condition is False. It's like a default path. 'If you have a VIP pass (if condition), enter the VIP area. *Else* (if you don't), proceed to the general admission area.' 

player_score = 500
if player_score >= 1000:
    print("You reached the high score!") # This condition is False
else:
    print("Keep trying to reach 1000 points.") # So, this alternative is executed.

The elif (Else If) Statement:

Allows you to check multiple conditions in sequence. If an if or elif condition is True, its block is executed, and the rest are skipped. An else block at the end is optional and runs if none of the preceding if or elif conditions were True. Think of it as sorting items into different bins based on criteria. Once an item fits a criterion and is placed in a bin, you don't check the other bins for that item. 

grade = 85

if grade >= 90: # Check for 'A' bin
    print("Grade: A")
elif grade >= 80: # Not an 'A', so check for 'B' bin. This is True.
    print("Grade: B") # Item placed in 'B' bin. Subsequent checks are skipped.
elif grade >= 70: # Skipped
    print("Grade: C")
else: # Skipped
    print("Grade: Needs Improvement")

4. Logical Operators: and, or, not

Logical operators are used to combine or modify boolean expressions, allowing for more complex decision-making. They help you set up more detailed rules for your conditional paths.

  • and: Returns True if both conditions are True. Think of this like needing multiple specific keys to open a treasure chest. If you need a 'blue key' (Condition 1) AND a 'square key' (Condition 2), having only one won't open it. Both must be true. (e.g., (5 > 3) and (10 == 10) is True because both 5 > 3 is true and 10 == 10 is true)
  • or: Returns True if at least one of the conditions is True. This is like a door that can be opened by one of several keys. If you have a 'master key' (Condition 1) OR a 'room-specific key' (Condition 2), either one (or both) will grant access. (e.g., (5 < 3) or (10 == 10) is True because 5 < 3 is false, but 10 == 10 is true)
  • not: Reverses the boolean value. It's like a flip switch for truth. If a condition is True, not makes it False. If it's False, not makes it True. So, not True is False, and not False is True. (e.g., not (5 > 3) is False because 5 > 3 is normally True)
age = 25
has_ticket = True

# Using 'and'
if age >= 18 and has_ticket:
    print("Welcome to the event!") # This will be printed
else:
    print("Access denied.")

# Using 'or'
is_weekend = True
is_holiday = False
if is_weekend or is_holiday:
    print("Enjoy your day off!") # This will be printed

# Using 'not'
is_raining = False
if not is_raining:
    print("It's a sunny day!") # This will be printed
